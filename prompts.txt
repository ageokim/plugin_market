project 제목 : plugin_market
컨셉 : claude에서 사용할 plugin을 관리하는 것이 목적
사용방법: pm.bin으로 경로 설정하여 어디서든 pm list,... 이런 식으로 사용
기능 :
0) list
-> 특정 github 조직에 있는 projec를 scan하여 description에 #plugin과 #release가 있는 것들을 load 및 list로 보여줌
-> load된 것들은 json 파일로 관리
-> json : {
    plugin name
    github addr
}
1) install/uninstall
-> install를 하면 0)에서 json에 저장된 plugin list를 보여줌
-> 사용자가 선택을 하면 git으로 다운받고 .claude에 등록(plugin_root_path, project_path 심볼링크)
-> uninstall 시 해당 git 삭제
2) enable/disable
-> enable은 .claude에 등록(project_path 심볼링크)
-> disable은 심볼링크 삭제
3) inspects
-> Plugin들의 상태 확인 (사용가능? enable?등등)

제약사항: plugin_market에서 Local claude가 동작, 즉 등록도 plugin_market/.cladue에 등록 및 관리가 되는 구조

file tree
plugin_market
 |- env : win/linux os에 따라 setup하는 것들을 모아둠
 |- plugins : 설치한 Plugin들의 project 관리
 |-.claude: local claude 관리
 |-CLAUDE.md
 |-docs: Architecture.md 관리
 |-README.md : User Manual처럼 관리
 |-Scripts: 동작파일들 관리
